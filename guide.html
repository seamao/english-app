📖 使用指南

Web3 商务英语学习工具 · 从 0 到上手 5 分钟

⚡ 一分钟上手

  1. 去 aistudio.google.com 免费申请 Gemini API Key（点 Get API Key → Create API Key）
  2. 回到本站，点右上角「设置」Tab，把 Key 粘到「Gemini API Key」输入框，点别处自动保存
  3. 回到「每日」Tab 开始练。就这么简单。

📱 手机装机（强烈推荐）

iPhone Safari：打开本站 → 底部中间分享按钮 → 向下滑 → 「添加到主屏幕」。之后像原生 App 一样全屏用，没有浏览器地址栏。

Android Chrome：右上角菜单 → 「添加到主屏幕」。

🎯 怎么用每个 Tab

每日

每天打开先看这里。系统自动挑 3 个词块 + 2 个句型 + 1 个场景，到期复习优先（用的是 Anki 那套 SM-2 算法，学得越熟间隔越长）。每项学完打勾，明天不会再推同样的。

句型库

30 个 BD 万能句型，每个配 4 个真实 Web3 例句。比如「We're looking to...」「Can we circle back on...」。点朗读能听带情感的 TTS（Gemini 合成的，比浏览器自带好听）。

场景对练（核心功能）

40 个 Web3 BD 真实场景，三层难度：

  • L1 跟读：看完整范文，逐句听原声 + 跟读录音，对比发音
  • L2 填空：给你模板和关键字段，你填具体内容（比如项目名、金额、日期）
  • L3 自由生成：只给提示词和必用短语，你全英文写/说，Gemini 批改

场景覆盖：交易所 listing DM、VC 冷启动邮件、AMA 开场、融资谈判、tokenomics 解释、做市商对接、项目危机公关、会议主持、pitch deck 讲解⋯⋯

发音

8 大亚洲人英语发音坑（θ / ð / v / w / r / l / æ / ɪ）。每个音给口型说明 + 高频词 + 录音评分。录完 Gemini 多模态直接给分数 + 逐词音标对比，知道哪个音没发对。

词块库

50+ Web3 高频短语（listing、tokenomics、TVL、rug pull、marketcap 之类），带音标 + 中文 + 真实例句。可以按类别筛选，标记已学/易错。

错题本

AI 批改发现的错误自动进这里。可以导出 Markdown，粘回 Obsidian 或笔记软件。

实战素材

拉当天 Twitter 热门推文（需填 SocialData Key，socialdata.tools）+ Reddit Web3 板块帖子。点任一条，Gemini 基于这条真实内容给你生成 3 种难度的写作任务 —— 简单版：复述一句；中等版：写条 BD DM；困难版：写 3 分钟 AMA 开场白。

☁️ 跨设备同步（可选）

如果你想手机 + 电脑进度互通，去「设置」→ 滑到「跨设备同步」：

  1. Worker URL 填：https://web3en-sync.exchangemhb.workers.dev（这是作者的，免费共享）
  2. 第一台设备点「生成 Key」→ 会出一串随机字符 → 复制
  3. 第二台设备填同样的 Worker URL，粘同一个 Key
  4. 之后每次改动 3 秒内自动上云，打开另一台设备自动拉下来

Key 就是密码，别给陌生人 —— 谁有 Key 谁就能读写你的数据。

如果你想完全独立部署自己的同步后端，看 GitHub README。Cloudflare 免费额度 10 万请求/天，个人用永远够。

❓ 常见问题

不填 Gemini Key 能用吗？

能。词块、句型、场景 L1/L2、发音训练全部可用，只有"AI 批改"、"Gemini TTS"、"发音评分"需要 Key。

Key 会不会泄露？

Key 只存你自己的浏览器 localStorage，不上传任何服务器（包括作者的）。所有代码开源，可以 在 GitHub 审计。

Gemini 免费额度够吗？

免费版 Gemini 2.0 Flash 每天 1500 次请求。一天学 30 分钟大概消耗 50-100 次，一个月都用不完。

数据会丢吗？

存在浏览器 localStorage。清浏览器数据会清掉。建议：1) 去「设置」定期「导出 JSON 备份」；2) 配跨设备同步，数据在云端有一份。

可以分享给朋友吗？

随便分享本站链接。但 不要把你的 Device Key 给朋友（会共享进度）。让他自己点「生成 Key」即可。
